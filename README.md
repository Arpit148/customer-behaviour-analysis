## 📌 Project Overview

This project analyzes customer shopping behavior using Python, Microsoft SQL Server (MSSQL), and Power BI. The objective is to transform raw transactional data into meaningful business insights through data cleaning, exploratory analysis, SQL querying, and interactive dashboards.

The project follows an end-to-end data analytics workflow, from data preprocessing to visualization, helping businesses understand customer purchasing patterns, product performance, and revenue trends.

---

## 📂 Dataset

The dataset contains customer shopping transactions with information such as:

- Customer ID
- Age
- Gender
- Location
- Item Purchased
- Category
- Purchase Amount
- Season
- Subscription Status
- Shipping Type
- Discount Applied
- Review Rating
- Previous Purchases
- Purchase Frequency

**Dataset Size**

- 📄 Rows: **3,900**
- 📊 Columns: **18**

# 🔍 Key Insights

- Clothing category generated the highest revenue.
- Young adults contributed the highest sales.
- Loyal customers accounted for the majority of purchases.
- Express shipping customers had a slightly higher average purchase value.
- Discount campaigns significantly influenced purchasing behavior.
- Subscription customers showed consistent shopping patterns.

# ▶️ How to Run

### 1 Clone Repository

```bash
git clone https://github.com/yourusername/customer-shopping-behavior-analysis.git
```

### 2 Install Libraries

```bash
pip install pandas matplotlib numpy
```

### 3 Run Python Notebook

Open

```
Customer_Analysis.ipynb
```

and execute all cells.

### 4 Import Data into Microsoft SQL Server

- Create a database.
- Import the cleaned CSV file.
- Execute the SQL queries provided in the `SQL` folder.

### 5 Open Power BI Dashboard

Open

```
Customer_Dashboard.pbix
```

Refresh the data source if necessary.
